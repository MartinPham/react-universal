REACT UNIVERSAL
===
*made with Recioto della Valpolicella*

* Browser || Native || Server
* Without **eject**
* Create React App 2.0 & Expo
* Redux (with injector)
* Saga (with injector)
* Reselect
* ImmutableJS
* React Router
* React Frontload



**Browser**

```
yarn start
```

**Native**

```
expo start
```

**Server**

```
yarn build && yarn serve
```
